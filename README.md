- **Core:** 강의 동영상 데이터를 기반으로 AI가 핵심 내용을 요약하고 관련 평가 문제를 자동 생성하여, 수강생에게 맞춤형 복습 환경을 제공하는 지능형 LMS 플랫폼

  - 동영상 콘텐츠 및 생성된 대규모 문제 데이터셋의 정밀한 관리를 위한 **Spring Boot 백엔드 비즈니스 로직 및 MySQL 스키마 설계**
