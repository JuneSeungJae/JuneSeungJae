### 🎓 LMS_Project_EduSys
> "교육 콘텐츠의 안정적인 전달과 체계적인 학습 이력 관리를 위한 교육 관리 시스템"

- **Core:** 웹 기반 학습 관리 시스템(LMS)으로, 강의 등록부터 수강생의 학습 진행률 추적까지 통합 관리하는 교육 시스템
- **Tech:** Java, Spring Boot, MySQL, Git
- **Key Highlights:**
  - 대규모 사용자 요청을 고려한 **관계형 데이터베이스(MySQL) 스키마 설계 및 인덱스 최적화**
  - 강사와 수강생 간의 권한 분리(RBAC) 및 보안 강화를 위한 **인증/인가 프로세스 구축**
  - 복잡한 강의 커리큘럼 데이터와 수강 이력의 정밀한 조회를 위한 **비즈니스 로직 및 API 구현**

  - 미착용 상태 감지 시 이벤트 데이터 실시간 적재 및 관리자 알림을 위한 **효율적인 백엔드 데이터 흐름 제어**

### 📅 PMS-Team-Project
> "Notion 스타일의 협업 워크스페이스 및 프로젝트 자원 분할 공유 시스템"

- **Core:** 가입된 유저들이 자유롭게 프로젝트 스페이스를 생성하고, 팀원을 초대하여 마일스톤 관리 및 작업 분할·공유를 실시간으로 진행하는 웹 프로젝트
- **Tech:** Java, Spring Boot, MySQL, React, Tailwind CSS
- **Key Highlights:**
  - 다대다(N:M) 협업 구조를 유연하게 지원하는 **Entity-Relationship Diagram (ERD) 구조 설계**
  - 프로젝트별 멤버 초대, 작업(Task) 할당 및 상태 전이를 추적하는 **마일스톤 중심의 스케줄링 API 개발**
  - 사용자의 시각적 몰입감과 실시간성 워크플로우를 보조하기 위한 **모던 프론트엔드 컴포넌트 구조화**
