- **Core:** 사내/폐쇄망 환경에서도 안정적으로 동작하는 독립형 프로젝트 및 멤버 관리 시스템
- **Tech:** Java, Spring Boot, MySQL, Git
- **Key Highlights:**

  - 권한 분리 기반의 멤버 관리 기능 및 마일스톤 중심의 **일정(Schedule) 추적 API 구현**
  - 외부 클라우드 의존도를 낮추고 로컬/인트라넷 환경에서 무중단으로 신뢰할 수 있는 데이터 서버 환경 지향

### 🎮 Roguelike UI Project
> "사용자 몰입감을 극대화하는 컴포넌트 기반의 모던 웹 인터페이스"

- **Core:** 판타지 로그라이크 세계관(Tower of Winter 기반)을 웹으로 이식한 인터랙티브 UI
- **Tech:** React, Tailwind CSS, Vite
- **Key Highlights:**
  - **Vite 번들러 기반**의 빠른 개발 환경 세팅 및 상태 변화가 잦은 게임 요소의 효율적 렌더링 최적화
  - 재사용성을 극대화하기 위한 가독성 높은 **Atomic 컴포넌트 단위 분리 및 구조화**
  - **Tailwind CSS**를 활용하여 다크 판타지 컨셉에 맞는 완성도 높은 반응형 테마 스타일링 레이아웃 구현

  - 유저 편의성을 위한 월별 자산 변동 데이터 통계 분석 알고리즘 설계

- **Core:** CIFAR-10 데이터셋 분석을 기반으로 성능을 고도화한 딥러닝 이미지 분류 실험

- **Key Highlights:**

  - 실험 데이터 반복 검증을 통한 학습 곡선(Loss/Accuracy) 분석 및 모델의 정확도 최적화 성과 도출
